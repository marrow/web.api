# Security Policy

## Supported Versions

The following versions of this package are actively maintained and the target of priority security-related patching.

| Version   | Supported          |
| --------- | ------------------ |
| `~ 1.0.0` | :white_check_mark: |
| `< 1.0`   | :x:                |
